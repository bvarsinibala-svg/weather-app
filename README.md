# Weather App

A simple and responsive Weather Application built using HTML, CSS, and JavaScript. The application fetches real-time weather data using the OpenWeatherMap API.

## Features

- Search weather by city name
- Display current temperature
- Show humidity levels
- Show wind speed
- Display weather conditions
- Responsive and user-friendly interface

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Enter a city name and get real-time weather information.
